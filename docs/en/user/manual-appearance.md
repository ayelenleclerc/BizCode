# User Manual: Appearance (light / dark theme)

## Where to change the theme

At the **bottom of the left sidebar**, there is a button whose label describes the available action (e.g. switch to light or dark theme, depending on the selected language).

## Behaviour

- The choice is **stored on this device** (persisted when you reopen the application).
- If you have never changed the theme, the app defaults to **dark mode**.
