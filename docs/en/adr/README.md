# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for BizCode.
Each ADR documents a significant technical decision: the context, the options considered, the decision made, and the consequences.

Format: lightweight [Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions) style.
