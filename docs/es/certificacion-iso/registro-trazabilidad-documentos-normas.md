# Registro de documentos — referencia normativa (indicativa)

**Documento:** QMS-DR-001  
**Versión:** 0.1  
**Fecha:** 2026-04-01  

Este registro enlaza **códigos de documentos controlados** (GOV/RSK/…/PROC-MAN) con **cláusulas indicativas** para preparación de auditoría. **No** sustituye asesoramiento formal de certificación.

| Código | Cláusulas indicativas (ejemplos) | Notas |
|--------|----------------------------------|-------|
| GOV-* | ISO 9001:2015 §4–§10 | Ver [trazabilidad-iso.md](trazabilidad-iso.md) |
| RSK-* | ISO/IEC 27001:2022 §6.1.2; ISO 9001 §6.1; ISO/IEC 42001 | Completar tratamiento en stubs RSK |
| SEC-* | ISO/IEC 27001:2022 Anexo A | El alcance SGSI puede ser limitado |
| QLT-* | ISO 9001 §8.3–§8.6 | Enlazar a calidad y CI |
| REQ-* | ISO 9001 §8.3; ISO/IEC/IEEE 29148 | Narrativa canónica en `docs/*/specs/` |
| TST-* | ISO 9001 §8.7; ISO/IEC/IEEE 29119 | Pruebas en `tests/` |
| ARC-* | ISO 9001 §8.1; controles 27001 | OpenAPI en `docs/api/openapi.yaml` |
| SRV-* | ISO/IEC 20000-1 | Según operación de servicio |
| HR-* | ISO 9001 §7.1.2; ISO/IEC 27001 §7.2 | Competencia |
| PRV-* | ISO/IEC 27701 | Alinear con `mapa-datos-personales.md` |
| AI-* | ISO/IEC 42001 | Si IA está en alcance |
| PROC-MAN-* | Vista de proceso; alinear con GOV/QLT | Ver [../processes/indice.md](../processes/indice.md) |
