# Glosario

## Términos de dominio (Argentina)

### A

**ARCA** — *Agencia de Recaudación y Control Aduanero* (sucesora de la ex AFIP).

**Artículo** — Producto vendido; código, descripción, precios, condición IVA, stock.

### C

**CF (Consumidor final)** — Cliente sin discriminación de IVA en la factura; Factura B.

**Condición IVA (cliente)** — RI, Monotributo, CF o Exento; define el cálculo en la factura.

**Condición IVA (artículo)** — 21%, 10,5% o exento.

**CUIT / CUIL** — Identificador fiscal argentino (11 dígitos, dígito verificador).

### F

**Factura A** — Para RI; IVA discriminado.

**Factura B** — Para CF/Mono; IVA incluido en el precio.

### M

**Monotributo** — Régimen simplificado argentino.

### N

**Neto** — Base imponible antes de IVA.

### R

**RI (Responsable Inscripto)** — Contribuyente inscripto en IVA.

**Rubro** — Categoría de producto.

### U

**U. Medida** — Unidad de medida (U, kg, l, etc.).
