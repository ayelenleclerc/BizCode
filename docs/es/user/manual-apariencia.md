# Manual de Usuario: Apariencia (tema claro / oscuro)

## Dónde cambiar el tema

En el **menú lateral izquierdo**, al pie, hay un botón cuyo texto indica la acción disponible (por ejemplo, pasar a tema claro o a tema oscuro, según el idioma configurado).

## Comportamiento

- La elección se **guarda en este equipo** (misma sesión al volver a abrir la aplicación).
- Si nunca cambió el tema, la aplicación usa por defecto el **modo oscuro**.
