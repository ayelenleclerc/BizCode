# Registro de documentos — referência normativa (indicativa)

**Documento:** QMS-DR-001  
**Versão:** 0.1  
**Data:** 2026-04-01  

Este registro liga **códigos de documentos controlados** (GOV/RSK/…/PROC-MAN) a **cláusulas indicativas** para preparação de auditoria. **Não** substitui assessoria formal de certificação.

| Código | Cláusulas indicativas (exemplos) | Notas |
|--------|----------------------------------|-------|
| GOV-* | ISO 9001:2015 §4–§10 | Ver [rastreabilidade-iso.md](rastreabilidade-iso.md) |
| RSK-* | ISO/IEC 27001:2022 §6.1.2; ISO 9001 §6.1; ISO/IEC 42001 | Preencher tratamento nos stubs RSK |
| SEC-* | ISO/IEC 27001:2022 Anexo A | Escopo SGSI pode ser limitado |
| QLT-* | ISO 9001 §8.3–§8.6 | Ligado a qualidade e CI |
| REQ-* | ISO 9001 §8.3; ISO/IEC/IEEE 29148 | Narrativa canônica em `docs/*/specs/` |
| TST-* | ISO 9001 §8.7; ISO/IEC/IEEE 29119 | Testes em `tests/` |
| ARC-* | ISO 9001 §8.1; controles 27001 | OpenAPI em `docs/api/openapi.yaml` |
| SRV-* | ISO/IEC 20000-1 | Conforme operação de serviço |
| HR-* | ISO 9001 §7.1.2; ISO/IEC 27001 §7.2 | Competência |
| PRV-* | ISO/IEC 27701 | Alinhar a `mapa-dados-pessoais.md` |
| AI-* | ISO/IEC 42001 | Se IA estiver no escopo |
| PROC-MAN-* | Visão de processo; alinhar a GOV/QLT | Ver [../processes/indice.md](../processes/indice.md) |
