# Manual do usuário: Aparência (tema claro / escuro)

## Onde alterar

No **rodapé da barra lateral**, há um botão com o texto indicando a ação (conforme o idioma da interface).

## Comportamento

- A escolha é **salva neste equipamento**.
- Se nunca alterou, o padrão é o **modo escuro**.
