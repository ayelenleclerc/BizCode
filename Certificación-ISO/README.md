# Certificación-ISO — punto de entrada

Esta carpeta es el **punto de entrada** para la documentación orientada a auditoría (**ISO-ready**). El contenido controlado vive en **un solo lugar** por idioma bajo `docs/{en,es,pt-br}/certificacion-iso/`; aquí no se duplica el cuerpo de los documentos.

| Idioma | Índice del paquete (registro maestro y códigos) |
|--------|--------------------------------------------------|
| English | [docs/en/certificacion-iso/iso-package-index.md](../docs/en/certificacion-iso/iso-package-index.md) |
| Español | [docs/es/certificacion-iso/indice-paquete-iso.md](../docs/es/certificacion-iso/indice-paquete-iso.md) |
| Português (Brasil) | [docs/pt-br/certificacion-iso/indice-pacote-iso.md](../docs/pt-br/certificacion-iso/indice-pacote-iso.md) |

**Política:** no se afirma certificación ISO en el repositorio; se mantiene trazabilidad y evidencia según [docs/I18N_DOCUMENTATION.md](../docs/I18N_DOCUMENTATION.md) y [docs/DOCUMENT_LOCALE_MAP.md](../docs/DOCUMENT_LOCALE_MAP.md).

**Calidad operativa** (estrategia de pruebas, CI/CD, plan Swagger/OpenAPI) permanece en `docs/*/quality/` y se enlaza desde el índice del paquete.
